### 踩坑记录：飞书文档 API block_type

飞书文档 API 的 block_type 编号和文档说明有出入，实测结果：
- `block_type: 2` + key `text` → 普通文本 ✅
- `block_type: 4` + key `heading2` → 二级标题 ✅
- `block_type: 5` + key `heading3` → 三级标题 ✅
- `block_type: 12` + key `bullet` → 无序列表 ✅（不是 type 7！）
- `block_type: 22` + key `divider` → 分割线 ✅（不是 type 14！）
- 所有 text 类 block 都需要 `"style": {}` 字段，否则报 invalid param
- 批量创建最多 50 个 block/次
- 批量删除用 `blocks/{page_id}/children/batch_delete`（DELETE 方法，传 block_ids + start_index + end_index）
