## 创业帝国

### Zip2 (1995-1999)

| 年份 | 事件 |
|------|------|
| 1995 | 与弟弟 Kimbal 和 Greg Kouri 创立 Zip2（在线城市指南/黄页） |
| 1995-1999 | 为纽约时报、芝加哥论坛报等提供互联网目录服务；Musk 试图当 CEO 被董事会否决 |
| 1999.02 | Compaq 以 $3.07 亿现金收购 Zip2，Musk 获得 $2200 万（7%股份） |

### X.com / PayPal (1999-2002)

| 年份 | 事件 |
|------|------|
| 1999 | 联合创立 X.com（在线金融服务），首批联邦保险在线银行之一 |
| 1999 末 | 投资人认为 Musk 经验不足，CEO 被换为 Intuit 前 CEO Bill Harris |
| 2000 | X.com 与 Confinity（Thiel & Levchin 创立，旗下有 PayPal）合并；Musk 重任 CEO |
| 2000 | ⚡ 因坚持用 Microsoft 技术而非 Unix，引发内部分裂；董事会再次罢免 Musk，Thiel 接任 CEO |
| 2001 | 公司更名为 PayPal |
| 2002 | eBay 以 $15 亿收购 PayPal；Musk 作为最大股东（11.72%）获得 $1.758 亿 |
| 2002 | 获得美国公民身份 |
