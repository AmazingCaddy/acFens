# Agent Prompt模板与信息源策略

## Agent Prompt 结构

spawn subagent时（`sessions_spawn`），用以下结构：

### Agent 1: 著作

```
你的任务：调研[人名]的著作和系统性长文。

搜索方向：
- 出版书籍（书名、核心论点、出版年份）
- 长篇newsletter/博客/论文
- 反复出现≥3次的核心论点（真信念）
- 自创术语和概念
- 推荐书单（揭示智识谱系）

输出要求：
- 写入 [skill目录]/references/research/01-writings.md
- 每条信息标注来源URL和可信度
- 区分一手 vs 二手
- 矛盾直接记录，不调和

信息源黑名单：不使用知乎、微信公众号、百度百科。
```

### Agent 2: 对话

```
你的任务：调研[人名]的播客、长视频、AMA、深度采访。

搜索方向：
- 被追问时的回答方式
- 即兴类比
- 改变立场的瞬间
- 拒绝回答的问题
- 长对话中的思维过程暴露

输出：[skill目录]/references/research/02-conversations.md
```

### Agent 3: 表达DNA

```
你的任务：调研[人名]的碎片表达——Twitter/X、微博、即刻、短文。

搜索方向：
- 高频用词和句式
- 争议立场
- 幽默方式
- 公开辩论和论战

输出：[skill目录]/references/research/03-expression-dna.md
```

### Agent 4: 他者视角

```
你的任务：调研他人对[人名]的分析、评价、批评。

搜索方向：
- 外部观察到的行为模式
- 批评与争议
- 与同行的对比分析
- 传记/profile中的描述

输出：[skill目录]/references/research/04-external-views.md
```

### Agent 5: 决策记录

```
你的任务：调研[人名]的重大决策、转折点、争议行为。

搜索方向：
- 决策背景与逻辑
- 事后反思
- 言行一致/不一致案例
- 关键取舍

输出：[skill目录]/references/research/05-decisions.md
```

### Agent 6: 时间线

```
你的任务：整理[人名]从出生/出道到现在的完整时间线。

搜索方向：
- 关键里程碑
- 思想转折点
- 最近12个月动态（防过时）

输出：[skill目录]/references/research/06-timeline.md
```

## 信息源优先级

| 来源类型 | 揭示什么 | 权重 |
|---------|---------|------|
| 用户提供的一手素材 | 完整原文，未经过滤 | **最高+** |
| 本人著作 | 系统性思考 | 最高 |
| 长对话/访谈 | 即兴思维过程 | 最高 |
| 实际决策记录 | 真实行为 vs 声称 | 最高 |
| 社交媒体 | 表达风格、即时反应 | 中等 |
| 他人评价 | 外部视角、盲点 | 中等 |
| 二手转述 | 参考但需验证 | 低 |

## 本地语料处理

| 素材类型 | 处理方式 | 覆盖维度 |
|---------|---------|---------|
| 书籍PDF | 直接阅读提取核心论点 | 著作(01)、表达(03) |
| 演讲/访谈transcript | 分析问答模式和即兴反应 | 对话(02)、表达(03) |
| 视频字幕SRT | 同transcript处理 | 对话(02)、表达(03) |
| 博客/newsletter导出 | 提取系统性观点 | 著作(01)、表达(03) |
| 社交媒体导出 | 分析碎片表达模式 | 表达(03) |
| 内部文档/备忘录 | 分析决策逻辑 | 决策(05) |
